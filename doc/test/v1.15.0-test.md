# v1.15.0 测试文档 — 本地 Bot (Mastra 集成)

## 测试范围

v1.15.0 实现了本地 Bot 功能，通过 Mastra AI 框架在 Electron 主进程中运行 AI Agent：
1. **本地 Bot 运行模式** — 创建 Bot 时新增"本地运行"选项，配置 Mastra Provider/Model/API Key
2. **Mastra Agent 集成** — Electron 端集成 @mastra/core Agent + @ai-sdk/* 模型，支持流式输出
3. **流式消息渲染** — ChatWindow 中实时渲染 Local Bot 的打字机效果流式回复
4. **Mastra Tool 桥接** — 现有 Skill handler 自动包装为 Mastra createTool() 格式
5. **Socket.IO 流式中继** — Electron → Server → Client 三段式流式数据传递
