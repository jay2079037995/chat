# v1.12.0 测试文档 — 插件化 Skill + Bot 信任机制

## 测试范围

v1.12.0 实现了两大功能：
1. **可装卸 Skill** — 支持完整插件包（manifest.json + handler.js），服务端动态注册/注销/启用/禁用
2. **Bot 级信任** — 受信任 Bot 的所有 Skill 调用自动放行（含 dangerous），不受信任保持现有行为
