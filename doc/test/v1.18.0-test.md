# v1.18.0 测试文档 — 本地 Bot 支持市场自定义 Skill

## 测试范围

v1.18.0 实现了以下功能：
1. **MastraToolBridge 接入 SkillPackageManager** — `getAvailableMastraTools()` 和 `listMastraToolInfo()` 合并内置 + 自定义 Skill
2. **LocalBotManager 配置获取** — 新增 `getConfig(botId)` 方法，支持重建 Agent
3. **Skill 安装/卸载后自动重建** — main.ts 中安装/卸载 Skill 后，自动重建所有活跃本地 Bot 的 Tool 列表
