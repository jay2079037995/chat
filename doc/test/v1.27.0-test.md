# v1.27.0 测试文档 — 本地机器人自定义工作目录 + 机器人 ID 显示

## 手动测试

### 自定义工作目录
- [ ] 创建本地机器人 → 默认工作目录为 `{userData}/bots/{botId}/workspace`
- [ ] 编辑弹窗 → 点击"选择"按钮 → 选择自定义目录 → 工作目录路径更新
- [ ] 发消息给 Bot → Bot 使用 bash_exec 执行 `pwd` → 输出为自定义目录
- [ ] 编辑弹窗 → 点击"默认"按钮 → 恢复默认工作目录
- [ ] 点击"打开"按钮 → 在文件管理器中打开当前工作目录
- [ ] 重启 Electron 应用 → 自定义工作目录设置保持不变

### 机器人 ID 显示
- [ ] 机器人列表 → 每个 Bot 显示 ID 前 8 位
- [ ] 点击 ID 旁的复制按钮 → 复制完整 Bot ID
