# v1.17.0 测试文档 — Skill 安装同步修复 + 文件上传增强

## 测试范围

v1.17.0 修复了以下问题：
1. **Skill 安装同步** — Skill 市场安装新 Skill 后，Bot 配置页立即可见（通过 `onSkillChanged` 回调触发 `syncSkillsToServer` 同步到服务端）
2. **文件上传大小限制** — `MAX_FILE_SIZE` 从 10MB 提升至 1GB，解决大文件上传 413 错误
3. **Skill 注册表 URL** — 修正默认注册表 URL 为正确的 GitHub 仓库地址
