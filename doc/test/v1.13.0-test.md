# v1.13.0 测试文档 — DeepSeek 推理模型 + Bot Skill 定制

## 测试范围

v1.13.0 实现了两大功能：
1. **DeepSeek reasoner 模型接入** — 支持 deepseek-reasoner（R1 推理模型），完整展示思维链推理过程
2. **Bot Skill 定制** — 用户可为每个 Bot 选择可用的 Skill，实现精细化 Bot 能力管控
