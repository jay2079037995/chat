# v1.14.0 测试文档 — LLM 调用日志 + Skill 市场

## 测试范围

v1.14.0 实现了两大功能：
1. **LLM 调用日志** — Server Bot 管理界面可查看每次 LLM API 调用的完整日志（请求/响应/错误/耗时）
2. **Skill 市场** — 在线浏览/搜索/安装/卸载自定义 Skill，支持注册表管理（仅 Electron 桌面端）

另外修复了一个预存 bug：webpack-dev-server 开发模式下 publicPath 配置导致浏览器白屏。
