# v1.16.0 测试文档 — Skill 系统标准化（Claude Plugins 兼容）

## 测试范围

v1.16.0 将 Skill 系统从自定义 manifest.json 格式迁移至 SKILL.md 标准（兼容 Claude Plugins / Agent Skills Open Standard）：
1. **SKILL.md 标准格式** — 使用 YAML frontmatter + Markdown 指令替代 manifest.json
2. **SkillMdParser** — 新增 SKILL.md 解析器，解析 frontmatter 元数据和 Markdown 指令
3. **SkillPackageManager 改造** — 读取 SKILL.md 替代 manifest.json，handler 路径改为 scripts/handler.js
4. **Git 安装支持** — SkillMarketplace 支持从 Git 仓库（含 GitHub tree URL 子目录）安装 Skill
5. **内置 Skill 元数据扩展** — 所有 10 个内置 Skill 增加 version、license、author、tags 字段
6. **前端 UI 改造** — Skill 市场增加 Git 安装入口、Skill 详情弹窗展示 SKILL.md 内容
